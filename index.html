Projeto Login

Projeto desenvolvido para treinar a implementação de uma interface de login responsiva.

Saiba mais...

O projeto foi desenvolvido com foco em praticar HTML5 e CSS3, especialmente na criação de uma interface de login moderna e responsiva. A proposta foi simular uma tela real de autenticação, com campos de e-mail e senha, validações básicas e adaptação para diferentes tamanhos de tela, como celular, tablet e desktop.

Durante o desenvolvimento, foram aplicados conceitos como Flexbox, media queries e organização semântica do código. Também houve atenção à experiência do usuário, buscando um layout limpo, intuitivo e visualmente agradável.

Embora o projeto tenha seguido as orientações propostas pelo professor, foram realizadas adaptações na imagem utilizada e na paleta de cores escolhida, com o objetivo de personalizar a interface. Esse projeto serviu como base para consolidar conhecimentos fundamentais de front-end e exercitar a criatividade dentro de uma proposta orientada.
